# Changelog

### Added
- Novo projeto em pasta dedicada `tatersal-digital-ws-2`
- Overlay responsiva para OBS Browser Source
- WebSocket padrão `ws://localhost:3000`
- Servidor local padrão porta `8080`
- Reconexão com backoff exponencial + jitter
- Validação de mensagens com Zod
- Suporte a heartbeat/ping
- Indicador visual de stale data
- Painel de fallback manual
- Scripts de start `.sh/.bat`
- Script de healthcheck
- Logs estruturados (Pino)
- Testes unitários para parsing/normalização
- Dockerfile + docker-compose opcionais
- Documentação operacional em português
